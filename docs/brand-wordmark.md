# SineDay wordmark

## Official asset

Use `/assets/brand/sineday-wordmark.svg` as the canonical SineDay wordmark. It is a transparent extraction of the custom serif lettering in the index hero artwork. `/assets/brand/sineday-wordmark-light.png` is the raster fallback.

Use the wordmark for:

- Dashboard and account headers
- Login and onboarding brand headers
- Marketing mastheads
- Email header graphics
- Social graphics
- App-store promotional artwork

Do not use the wordmark for:

- Paragraph mentions of SineDay
- Buttons such as “Open SineDay”
- Form labels
- Footer legal text
- Screen-reader-only text
- Browser page titles or metadata

## Supporting typography

Keep the existing system font for functional UI headings and body text. Use spaced uppercase text for small branded caps. Do not add a global decorative serif font to imitate the wordmark.

The visual hierarchy is:

1. Brand wordmark: the custom SineDay asset
2. UI headings: the existing system/SF-style font
3. Body text: the existing system/SF-style font
4. Small branded caps: spaced uppercase
